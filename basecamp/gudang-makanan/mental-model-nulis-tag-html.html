Mental Model Nulis Tag HTML

Ketika aku mau nulis tag HTML (<html>, <script>, dsb.), step-stepnya gini:

  1. Ketik dulu kurung lancipnya. < terus >. Jadi bentuknya kurungan gini: <>
  2. Terus arahin cursor text editor (garis vertical | yang biasanya kedip-kedip, tempat karakter yang kita ketik keluar) ke dalem kurungan lancip tadi. Jadi gini: <|>
  3. Terus ketik nama element HTML-nya, misal html atau script. Jadi gini: <html>, <script>

Kalau udah terbiasa, ga akan ada lagi tuh human error yang bikin kita lupa nutup tagnya kaya gini: <html, <script.

Berlaku juga untuk elemen HTML yang ada opening sama closing tag-nya.

  1. Ketik dulu opening tagnya: <html>
  2. Terus ketik closing tagnya: </html>. Jadi gini: <html></html>
  3. Arahin cursor text editor ke tengah-tengah burger HTML-nya. Gini: <html>|</html>
  4. Baru lanjutin ngetik isinya. Misal: <html>halo guys</html>
